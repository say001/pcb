# description
a totoro inspired custom pcb that has LEDs that activate when the push button is pressed. the LEDs slowly fade in and out due to capacitor.

# materials

- 2 LEDs
- 1 battery cell
- 3 resistors
- NPN transistor
- push button
- resistor
- capacitor
