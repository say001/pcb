# description
a totoro inspired custom pcb that has LEDs that activate when the push button is pressed. the LEDs alternate being on and fade out. 

# materials

- 2 LEDs
- 1 battery cell
- 2 resistors
- 2 NPN transistors
- push button
- 2 capacitors
